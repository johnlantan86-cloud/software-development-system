# Skill Laboratory 0.8 — Git Conflict Awareness

## Purpose
Learn what Git conflicts are and how to recognize conflict markers before fixing conflicts.

## Skills Practiced
- Understanding why conflicts happen
- Recognizing conflict markers
- Identifying current branch changes
- Identifying incoming branch changes

## Evidence
This lab is complete when:
- I can explain what a conflict means
- I can identify HEAD
- I can identify the incoming branch section
- I can explain why Git cannot decide automatically
