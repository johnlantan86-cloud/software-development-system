# Skill Lab 1.2 — HTML Text and Page Content Structure

## Purpose
Practice using HTML tags that give meaning to page content.

## Skills Practiced
- Creating a main page heading
- Creating section headings
- Writing paragraphs
- Creating unordered lists
- Using sections to group related content

## Evidence
This lab is complete when:
- The page has one `h1`
- The page has at least two `h2` headings
- The page has one paragraph
- The page has one unordered list
- I can explain the difference between `h1`, `h2`, `p`, `ul`, and `li`
