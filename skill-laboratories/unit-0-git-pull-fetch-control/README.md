# Skill Laboratory 0.7 — Git Pull and Fetch Control

## Purpose
Practice checking GitHub changes safely before bringing them into the local repo.

## Skills Practiced
- Using git fetch
- Using git pull
- Checking branch tracking
- Understanding local branch vs origin branch
- Checking whether local and GitHub match

## Evidence
This lab is complete when:
- I can explain git fetch
- I can explain git pull
- I can explain why fetch is safer when checking first
- I can confirm whether my branch is up to date with origin/master
