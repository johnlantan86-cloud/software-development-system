# Skill Lab 1.1 — HTML Document Structure

## Purpose
Practice creating a valid HTML document from scratch.

## Skills Practiced
- Using `<!DOCTYPE html>`
- Creating the `<html>`, `<head>`, and `<body>` sections
- Adding metadata
- Adding visible page content

## Evidence
This lab is complete when:
- The HTML file has a valid document structure
- I can explain the purpose of `head`
- I can explain the purpose of `body`
- I can open the page in a browser
