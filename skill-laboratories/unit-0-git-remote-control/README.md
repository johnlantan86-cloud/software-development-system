# Skill Laboratory 0.6 — Git Remote Control

## Purpose
Practice understanding and checking Git remotes before connecting to GitHub.

## Skills Practiced
- Checking remotes
- Understanding origin
- Understanding local repo vs remote repo
- Preparing to connect a local repo to GitHub

## Evidence
This lab is complete when:
- I can explain what a remote is
- I can explain what origin means
- I can check whether a repo has a remote
- I can safely prepare before pushing to GitHub
