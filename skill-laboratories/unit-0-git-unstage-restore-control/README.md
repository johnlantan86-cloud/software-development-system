# Skill Laboratory 0.4 — Git Unstage and Restore Control

## Purpose
Practice undoing staged and unstaged changes safely.

## Skills Practiced
- Staging files
- Unstaging files
- Restoring accidental edits
- Checking Git status after every action

## Evidence
This lab is complete when:
- I can unstage a file without deleting its content
- I can restore accidental edits
- I can explain the difference between unstage and restore
