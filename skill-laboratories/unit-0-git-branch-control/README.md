# Skill Laboratory 0.5 — Git Branch Control

## Purpose
Practice creating and working safely on a Git branch.

## Skills Practiced
- Checking the current branch
- Creating a new branch
- Switching branches
- Making commits on a branch
- Understanding that commits belong to the branch where they are created

## Evidence
This lab is complete when:
- I can create a branch
- I can confirm which branch I am on
- I can make a commit on a branch
- I can explain why branches protect the main version
