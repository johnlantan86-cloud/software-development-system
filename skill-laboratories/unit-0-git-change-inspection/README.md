# Skill Laboratory 0.3 — Git Change Inspection

## Purpose
Practice checking exactly what changed before staging and committing.

## Skills Practiced
- Using git status
- Using git diff
- Using git diff --name-only
- Checking the correct file before committing
- Avoiding accidental commits

## Evidence
This lab is complete when:
- I can explain what git diff does
- I can inspect file changes before committing
- I can commit only intended changes
