# Practice

## Command Meaning Practice

1. `pwd` shows the current folder location.
2. `dir` lists files and folders.
3. `mkdir folder-name` creates a folder.
4. `ni file-name.md -ItemType File` creates a file.
5. `git status` shows the current Git state.
6. `git add file-name` stages a file.
7. `git commit -m "message"` saves staged changes into Git history.
8. `git log --oneline` shows commit history in short form.
