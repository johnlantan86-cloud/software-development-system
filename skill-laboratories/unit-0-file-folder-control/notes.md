# Notes

## Key Rules

- mkdir creates folders.
- ni file-name -ItemType File creates files.
- pwd shows the current location.
- dir lists files and folders.
- Git tracks files, not empty folders.
- d----- means folder.
- -a---- means file.

## Personal Mistake Patterns

- I must not use mkdir when I want to create a file.
- I must remember that -ItemType needs the dash.
- I must use code . when I want to open the current folder.
- I must check whether something is a file or folder using d----- and -a----.
