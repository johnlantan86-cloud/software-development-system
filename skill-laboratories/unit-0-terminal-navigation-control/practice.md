# Practice

## Terminal Navigation Rules

1. `pwd` shows where I am.
2. `dir` shows what is inside the current folder.
3. `cd folder-name` enters a folder.
4. `cd ..` moves one folder back.
5. `code .` opens the current folder in VS Code.
6. I must check my location before creating files or folders.

## Personal Rule

Before creating anything, I will run:

`pwd`

Then:

`dir`
## Personal Navigation Rule

Before I create or edit files, I must confirm I am in the correct folder using `pwd` and `dir`.
