# Skill Laboratory 0.2 — Terminal Navigation Control

## Purpose
Practice knowing where I am in the terminal, moving between folders safely, and checking files before creating or editing anything.

## Skills Practiced
- Using pwd to confirm location
- Using dir to list files and folders
- Using cd to enter folders
- Using cd .. to move back
- Using code . to open the current folder
- Avoiding wrong-location mistakes

## Evidence
This lab is complete when:
- I can explain pwd, dir, cd, cd .., and code .
- I can move around the repo without getting lost
- I can verify my location before creating files
- The lab is committed in Git
